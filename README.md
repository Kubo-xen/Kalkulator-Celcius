# Kalkulator Celcius dengan C++

Kalkulator sederhana berbasis **C++** untuk mengonversi suhu dari **Celcius** ke:
- Reamur
- Fahrenheit
- Kelvin

Project ini cocok untuk pemula yang ingin memahami dasar input, output, dan perhitungan matematika di C++.
